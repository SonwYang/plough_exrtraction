# 1. 数据集准备

## 1.1. 数据制作

- 利用arcgis制作耕地面矢量；
- 面矢量转线矢量；
- 面矢量转栅格，线矢量转栅格；
- 耕地线膨胀两个像素后，叠加到耕地面上，其中背景的像素值为0，耕地面的像素值为1，耕地线为2。
